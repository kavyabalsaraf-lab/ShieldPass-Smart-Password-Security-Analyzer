# 🔐 ShieldPass Analyzer

A simple Python desktop application that analyzes password strength and securely encrypts passwords using **Fernet Encryption**.

## ✨ Features

- Check password strength in real time
- Generate strong random passwords
- Encrypt passwords before saving
- Show/Hide password
- Store email and encrypted password locally
- Simple and user-friendly Tkinter GUI

## 🛠️ Technologies Used

- Python
- Tkinter
- Cryptography (Fernet)
- Regular Expressions (re)

## 📁 Files

```
main.py
passwords.txt
secret.key
README.md
```

## ▶️ Run the Project

1. Install the required package:
```bash
pip install cryptography
```

2. Run the application:
```bash
python main.py
```
